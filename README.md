
# Karenderia Web App

## Project Overview
The **Karenderia Web App** is a simple web application designed to help manage daily operations of a small local eatery (karenderia).  
It allows users to view the menu, create orders, track daily sales, and provides a clean UI for easy management.

## Team Members & Roles

| Student | Role | Responsibilities |
|---------|------|-----------------|
| Student 1 | Team Leader | Create repository, prepare project structure & UI templates, review pull requests, test features, merge approved PRs |
| Student 2 | Menu Management | Implement menu display, add sample food items and prices |
| Student 3 | Order Management | Implement order creation, display selected items and total price |
| Student 4 | Daily Sales Summary | Display total sales and number of orders |
| Student 5 | UI & Documentation | Improve UI layout/styling, update README.md with instructions |
